⚠️ Unable to load required libraries

The application cannot load React and MUI libraries from CDN. This is usually caused by:

  • No internet connection - CDN requires internet access
  • Firewall/Proxy blocking CDN - Network restrictions
  • CDN service unavailable - Temporary issue

Solutions:

  1. Use offline version (recommended):
    • Run download-libs.bat in the frontend folder (requires one-time internet)
    • Then open index-offline.html
  2. Use local HTTP server:
    • Go to parent folder and run start-server.bat
    • Open http://localhost:8000
  3. Check network settings:
    • Ensure you have internet access
    • Try disabling VPN or proxy
    • Open browser console (F12) for detailed errors

Quick test: Open test.html to diagnose the issue.
